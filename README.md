# Digital Footprint Analyzer

An Android application that analyzes and evaluates a user's digital footprint risk based on installed applications, permissions, system settings, and usage statistics.

## Modules

### Member 1 – Data Collection
- Installed app scanner
- Permission extraction
- Location status check
- Bluetooth status check
- Usage statistics collection

### Member 2 – Risk Engine
- Rule-based risk score calculation

### Member 3 – UI / Dashboard
- Risk display interface
- Visualization of results

### Member 4 – Recommendations
- Personalized privacy suggestions
